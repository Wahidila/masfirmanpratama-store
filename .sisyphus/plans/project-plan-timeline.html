Konfigurasi Penting

  • Admin Panel: Route prefix /admin di Store app (bukan subdomain)
  • Skema Cicilan: Admin bisa set bebas dari panel (CRUD)
  • Rekening Tujuan: Admin bisa set multi-rekening dari panel
  • Affiliate Domain: affiliate.masfirmanpratama.com (subdomain Hostinger)
  • Queue Driver: Database (shared hosting, bukan Redis)
  • Cron Job: Via Hostinger panel (reminder cicilan, expire orders)
  • Deploy: SSH + Composer + artisan migrate
